Phone Gap

What is Phone Gap

A platform that allows app development only from front-end.
Only requires JS and HTML to make phone apps.
One phone gap app is compatible with all OS.
Developed by Adobe.


Requirements

Phonegap Build on desktop.
Phonegap app on mobile.
The app runs on a web server during development, and can be accessed on mobile.
Edit the code on desktop to see change reflected on mobile.
Build the app on build.phonegap.com (requires Adobe or github account)


Advantages

Convenient deployment.
Only requires HTML/JS.
API support of phone features - GPS, camera, etc.


Disadvantages

Slow, especially on canvas games.
Alternative platforms like cocoonJS
